## Overview

Customer 360 provides a **unified view** of all customer data across accounts, contacts, opportunities, cases, and interactions. It's the single source of truth for every customer relationship.

**Package:** `@hotcrm/crm` + `@hotcrm/support` + `@hotcrm/finance`

## Complete Customer View

### The Customer Data Graph

```
                   Account (Organization)
                          │
        ┌─────────────────┼─────────────────┐
        │                 │                 │
    Contacts         Opportunities        Cases

    ├─ Activities    ├─ Quotes         ├─ Comments
    ├─ Tasks         ├─ Contracts      ├─ Knowledge
    └─ Notes         └─ Invoices       └─ Escalations
```

## Integration & Data Flow

### Unified Customer Record

```
External Sources → HotCRM → 360° View
     ↓                ↓           ↓
 - CRM Data      - Dedupe    - Accounts
 - Support       - Enrich    - Contacts
 - Finance       - Score     - Opportunities
 - Marketing     - Relate    - Cases
 - Product       - Analyze   - Timeline
```

### Data Synchronization

- **Real-time**: Activity tracking, case updates
- **Hourly**: Health score recalculation
- **Daily**: Enrichment updates
- **Weekly**: Duplicate detection
