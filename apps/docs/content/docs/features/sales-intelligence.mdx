## Overview

HotCRM's Sales Intelligence combines **Customer 360** (unified customer data) with **AI-powered insights** to help sales teams close deals faster and more predictably.

**Package:** `@hotcrm/crm`  
**Objects:** 13 | **AI Actions:** 8 | **Hooks:** 7

## Customer 360: The Single Source of Truth

### Account Management

**Object:** `account.object.ts`

Enterprise-grade customer and organization management:

```typescript
// Account fields
{
  name: 'account',
  fields: [
    'name',                    // Organization name
    'account_number',          // Auto-generated
    'type',                    // Prospect, Customer, Partner, Competitor
    'industry',                // 15+ industries
    'annual_revenue',          // Currency field
    'number_of_employees',     // Number field
    'billing_address',         // Address field
    'shipping_address',        // Address field
    'parent_account_id',       // Self-reference for hierarchies
    'owner_id',                // User reference
    'health_score',            // AI-calculated (0-100)
  ]
}
```

**AI Features:**
- **Health Scoring**: Automated account health calculation based on activity, revenue, support cases
- **Risk Detection**: Churn prediction and early warning signals
- **Upsell Opportunities**: AI-identified expansion opportunities

### Contact Management

**Object:** `contact.object.ts`

Individual relationship tracking with decision-maker identification:

```typescript
// Contact fields
{
  name: 'contact',
  fields: [
    'first_name', 'last_name', 'email', 'phone',
    'title',                   // Job title
    'department',              // Functional area
    'decision_maker',          // Boolean
    'influence_level',         // High, Medium, Low
    'relationship_strength',   // Strong, Developing, Weak
    'account_id',              // Reference to Account
    'reports_to_id',           // Self-reference
    'email_opt_out',           // GDPR compliance
  ]
}
```

**Key Capabilities:**
- Decision maker identification
- Organizational hierarchy mapping
- GDPR-compliant communication preferences
- Relationship strength scoring

### Opportunity Pipeline

**Object:** `opportunity.object.ts`

AI-guided sales pipeline with 7-stage forecasting:

```typescript
// Opportunity fields
{
  name: 'opportunity',
  fields: [
    'name',                    // Deal name
    'account_id',              // Reference
    'amount',                  // Currency
    'close_date',              // Date
    'probability',             // AI-calculated
    'stage',                   // 7 stages
    'type',                    // New Business, Upsell, Renewal
    'lead_source',
    'next_step',               // Text
    'forecast_category',       // Pipeline, Best Case, Commit, Closed
  ]
}
```

**7-Stage Pipeline:**
1. **Prospecting** (10% probability)
2. **Qualification** (20%)
3. **Needs Analysis** (40%)
4. **Proposal** (60%)
5. **Negotiation** (80%)
6. **Closed Won** (100%)
7. **Closed Lost** (0%)

**AI Actions:**

#### 1. Win Probability (`opportunity_ai.action.ts`)
```typescript
// Calculates win probability based on:
- Historical win rates by stage
- Deal characteristics (size, type, source)
- Engagement metrics
- Competitive factors
- Time in stage

// Returns: Probability (0-100%) + Key factors
```

#### 2. Deal Risk Analysis
```typescript
// Identifies at-risk deals:
- Stalled opportunities (no activity)
- Slipped close dates
- Low engagement
- Missing decision makers

// Returns: Risk score + Recommended actions
```

#### 3. Revenue Forecasting
```typescript
// Forecasts pipeline revenue:
- By stage (weighted pipeline)
- By rep/team
- By time period
- Confidence intervals

// Returns: Forecast amounts + accuracy metrics
```

## Integration Capabilities

### API Access

```typescript
// Create an opportunity
POST /api/objects/opportunity
{
  "name": "ACME Corp - Enterprise Deal",
  "account_id": "acc_123",
  "amount": 250000,
  "close_date": "2026-06-30",
  "stage": "qualification"
}

// Score a lead
POST /api/actions/lead_scoring
{
  "lead_id": "lead_456"
}

// Get account health
GET /api/objects/account/acc_123?include=health_score
```

### Webhooks

Real-time notifications:
- `opportunity.stage_changed`
- `lead.converted`
- `account.health_score_changed`
- `activity.completed`
