## Prerequisites

Before installing HotCRM, ensure you have the following installed:

- **Node.js** 18.x or higher
- **pnpm** 8.x or higher (recommended package manager)
- **Git** for version control

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/objectstack-ai/hotcrm.git
cd hotcrm
```

### 2. Install Dependencies

HotCRM uses pnpm workspaces for monorepo management:

```bash
pnpm install
```

### 3. Build the Project

Build all packages in the correct dependency order:

```bash
pnpm build
```

### 4. Start the Server

Start the HotCRM application server:

```bash
cd packages/server
pnpm dev
```

The server will start on `http://localhost:3000` by default.

## Verify Installation

### Run Tests

Verify your installation by running the test suite:

```bash
# Run all tests
pnpm test

# Run tests for a specific package
cd packages/crm
pnpm test
```

### Check API Endpoints

Once the server is running, you can verify the API is working:

```bash
curl http://localhost:3000/api/health
```

## Configuration

### Environment Variables

Create a `.env` file in the `packages/server` directory:

```bash
# Database Configuration
DATABASE_URL=sqlite:./data/hotcrm.db

# Server Configuration
PORT=3000
NODE_ENV=development

# AI Service Configuration (optional)
OPENAI_API_KEY=your_api_key_here
```

### Database Initialization

The server will automatically initialize the SQLite database on first run. The database file will be created at `packages/server/data/hotcrm.db`.

## Troubleshooting

### Common Issues

**Issue: `pnpm` command not found**

Install pnpm globally:
```bash
npm install -g pnpm
```

**Issue: Build failures**

Clean the build cache and reinstall:
```bash
pnpm clean
rm -rf node_modules
pnpm install
pnpm build
```

**Issue: Port already in use**

Change the port in your `.env` file or use the PORT environment variable:
```bash
PORT=3001 pnpm dev
```
