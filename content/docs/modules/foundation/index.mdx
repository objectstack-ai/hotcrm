# Platform Foundation

The **Foundation** layer provides the shared services that power all HotCRM clouds. It ensures a unified view of the customer ("Customer 360") and provides the extension points for custom development.

## Core Capabilities

### 1. Customer 360
The single source of truth for customer data.
- **Account**: Business entities (B2B) or Households (B2C).
- **Contact**: Individuals associated with accounts.
- **Activity Timeline**: Unified view of emails, tasks, and meetings.

### 2. Identity & Security
- **SSO**: Single Sign-On integration.
- **Role Hierarchy**: Data visibility based on org chart.
- **Permission Sets**: Granular functional access.

### 3. Extensibility
- **ObjectQL**: The unified data access protocol.
- **Flow Engine**: No-code process automation.
