# Marketing Journeys

HotCRM Marketing is not just about sending emails; it's about orchestrating coherent **Customer Journeys** across every touchpoint.

## Journey Builder

The visual canvas for designing lifecycle campaigns.

*   **Triggers**: Start journeys based on data changes (e.g., "Status changed to Active," "Opened Support Ticket," "Birthday is today").
*   **Logic Branching**: Split paths based on customer behavior (e.g., "If clicked link -> Send SMS," "If no open -> Wait 3 days -> Resend").
*   **Multi-Channel**: Orchestrate Email, SMS, Push Notifications, and WhatsApp messages in a single flow.

## Audience Management

*   **Dynamic Segments**: create live lists based on complex criteria (e.g., "CEOs in FinTech who visited the pricing page yesterday").
*   **Lookalike Modeling**: Use AI to find new prospects that resemble your best existing customers.

## Campaign Intelligence

*   **Attribution Modeling**: First-touch, Last-touch, and Multi-touch attribution to understand which campaigns drive real revenue.
*   **Engagement Scoring**: Score leads based on email opens, clicks, and web activity to assist Sales prioritization.

## Content Studio

*   **Visual Editor**: Drag-and-drop email builder with responsive design.
*   **AI Copywriter**: Generate subject lines, body copy, and call-to-actions optimized for conversion.
*   **Personalization**: Insert dynamic ObjectQL data (e.g., `{{account.account_manager.name}}`) into any message.
