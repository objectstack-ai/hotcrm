## 🏗️ 架构创新 (Architectural Innovations)

### 插件化架构
```
@objectstack/runtime (核心引擎)
  ├── @hotcrm/crm (销售云)
  ├── @hotcrm/support (服务云)
  ├── @hotcrm/products (产品云)
  ├── @hotcrm/finance (财务云)
  ├── @hotcrm/hr (人力资源) ✅ 新增
  ├── @hotcrm/projects (项目管理) ✅ 新增
  ├── @hotcrm/inventory (库存管理) ✅ 新增
  ├── @hotcrm/prm (合作伙伴) ✅ 新增
  ├── @hotcrm/ai (AI服务层) ✅ 新增
  └── @hotcrm/marketplace (插件市场) ✅ 新增
```

### 元数据驱动
所有业务逻辑通过TypeScript元数据定义，无需手写CRUD代码：
```typescript
const Employee = {
  name: 'employee',
  fields: {
    employee_number: { type: 'autonumber' },
    first_name: { type: 'text', required: true },
    department_id: { type: 'lookup', reference: 'department' }
  }
};
```

### AI服务层
统一的AI服务抽象，支持多种LLM提供商：
```typescript
const aiService = new AIService({
  provider: 'openai', // or 'claude', 'huggingface'
  model: 'gpt-4'
});

const insights = await aiService.generateInsights(record);
```
