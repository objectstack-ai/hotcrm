## Core Design Principles

### 1. Metadata-First
Every object, field, view, and logic rule is defined as metadata (`.object.ts`, `.view.ts`). This ensures:
- **Upgradeability**: Customizations receive platform updates without breaking.
- **Portability**: Configurations can be moved between environments as code.
- **AI-Readiness**: AI agents can "read" the system configuration to understand how to operate.

### 2. ObjectQL (No-SQL)
We eliminated the impedance mismatch between code and data. No more ORMs or raw SQL.
- **Type-Safe Access**: Full TypeScript support for data operations.
- **Security-Aware**: Permissions are enforced at the engine level, not the query level.

### 3. AI-Native
AI is not a side panel. It is the UI.
- **Generative**: Write emails, summaries, and code.
- **Predictive**: Score leads, forecast churn, and estimate revenue.
- **Agentic**: Perform multi-step tasks autonomously (e.g., "Reschedule all meetings for next week").
