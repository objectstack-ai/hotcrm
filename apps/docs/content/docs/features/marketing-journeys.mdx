## Overview

HotCRM's Marketing Cloud provides AI-powered campaign management with intelligent content generation, multi-channel execution, A/B testing, and ROI tracking.

## Campaign Management

### Marketing Campaigns

**Object:** `campaign.object.ts`

Multi-channel campaign orchestration:

```typescript
{
  name: 'campaign',
  fields: [
    'name',
    'type',                // 9 channel types
    'status',              // Planned, In Progress, Completed, Aborted
    'start_date',
    'end_date',
    'budget_cost',         // Campaign investment
    'actual_cost',
    'expected_revenue',
    'actual_revenue',
    'expected_response',   // Expected response %
    'num_sent',            // Total sent
    'num_delivered',       // Successfully delivered
    'num_opened',          // Email opens
    'num_clicked',         // Link clicks
    'num_responded',       // Conversions
    'parent_campaign_id',  // Nested campaigns
  ]
}
```

**9 Campaign Types:**

| Type | Description | Use Case |
|------|-------------|----------|
| **Conference** | In-person events | Trade shows, conferences |
| **Webinar** | Online events | Product demos, training |
| **Email** | Email marketing | Newsletters, promotions |
| **Social Media** | Social campaigns | LinkedIn, Twitter, Facebook |
| **Advertisement** | Paid ads | Google Ads, display ads |
| **Direct Mail** | Physical mail | Catalogs, postcards |
| **Partners** | Channel partners | Co-marketing |
| **Trade Show** | Exhibition booths | Industry events |
| **Other** | Custom types | Any other channel |

### A/B Testing

**Built-in A/B test capabilities:**

```typescript
// Create A/B test
POST /api/campaigns/ab_test
{
  "campaign_id": "camp_123",
  "variants": [
    {
      "name": "Variant A",
      "subject_line": "Limited Time Offer",
      "traffic_percentage": 50
    },
    {
      "name": "Variant B", 
      "subject_line": "Exclusive Deal Inside",
      "traffic_percentage": 50
    }
  ],
  "winning_metric": "open_rate",
  "test_duration_hours": 24,
  "auto_select_winner": true
}

// After test completion:
{
  "winner": "Variant B",
  "results": {
    "variant_a": { "open_rate": 22.5, "sent": 5000 },
    "variant_b": { "open_rate": 28.3, "sent": 5000 }
  },
  "confidence_level": 95,
  "recommendation": "Roll out Variant B to remaining 90% of list"
}
```
